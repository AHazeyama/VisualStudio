<b>
* 指定した文字列を 
<span style="color:red">含まない</span>
ファイルを削除します。
</b> 
<br>
* 処理ディレクトリに子ディレクトリが存在する場合、ディレクトリに含まれるファイルに対して再帰的に削除する事が出来ます。  

<br>

<b><span style="color:red">注意!!</span></b>  
<b><span style="color:red">
本ツールは実行時に可否確認を行いません。ファイル/ディレクトリ関係無く、問答無用で削除します。
</span></b>

|項目|内容|
|:--|:--|
|Exec directory [ ]|処理対象ディレクトリ|
|not removed word [ ]|削除対象外文字列|
|☑ Recursive processing|ディレクトリ階層再帰処理|
|Processing message|処理内容表示ウィンドウ|
|[ExRemove]|削除実行|
|[Clear]|削除対象外文字列初期化|
|[Undo]|変換取り消し(初期状態に戻ります)|
|[Help]|Help表示|
|[Exit]|ツール終了|
