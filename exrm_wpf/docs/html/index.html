exrm_wpf

Executive remove tool [WPF] ReadMe

  • 指定した文字列を 含まない ファイルを削除します。
  • 処理ディレクトリに子ディレクトリが存在する場合、ディレクトリに含まれるファイルに対して再帰的に削除する事が出来ます。


注意!!
本ツールは実行時に可否確認を行いません。ファイル/ディレクトリ関係無く、問答無用で削除します。
