# Overview
### 業務効率化および運用改善を目的としたツール群。

ファイル操作や整合性検証といった反復作業を対象に、  
　* デスクトップアプリケーション ( **C#** [ **WPF** ] / **Python** [ **PySide6** / **Tkinter** ] )  
および  
　* Web API ( **Python** [ **FastAPI** ] )  
として実装しています。

単なる機能実装に留まらず、実運用を想定し、操作性・安全性・再現性を重視した設計としています。
